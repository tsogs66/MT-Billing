# MT-Billing Android Studio project

This folder is the native Android Studio wrapper generated by Capacitor for the MT-Billing web app.

## Open in Android Studio

1. From repo root, run:
   - `npm run android:sync`
2. Open Android Studio.
3. Choose **Open** and select this folder: `client/android`.
4. Wait for Gradle sync, then run:
   - **Build > Build APK(s)** for debug APK
   - **Build > Generate Signed Bundle / APK** for release builds

## What this project includes

- WebView shell that loads the full MT-Billing web app bundle (`client/dist`).
- Existing app ID and package:
  - `appId`: `com.tsogs.mtbilling`
  - Main activity: `com.tsogs.mtbilling.MainActivity`
- Capacitor plugins already wired (status bar, splash screen, keyboard, app lifecycle).

## Keep features in sync

Any web feature changes must be re-synced into Android:

- `npm run android:sync`

That command rebuilds the web app and copies updated assets into this Android project.
